В скрипте ratingsExtractor.py описана функция 
ratingsExtractor(actors, directors, writers) 
которая принимает списки имен актеров, режиссеров и сценаристов
функция возвращает кортеж ((actRat[0],...,actRat[n], dirRat, wrtRat), n) из двух значений
первое - кортеж из рейтингов актеров с рейтингом режиссера и сценариста в конце
второе - число рейтингов актеров
